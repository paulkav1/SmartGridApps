About These Apps


This tool set is a representative part of a larger group of tools that allows users to request, manage, and comprehend extracts from the several databases of smart grid data. In particular we are monitoring these devices which represent a hierarchy from the top of the distribution grid (feeder) to the bottom (premises):

  • Breaker
  • Automated Feeder Switch (AFS)
  • Automated Lateral Switch (ALS or Trip saver)
  • Fault Current Indicator
  • Smart Meter

Feature Set

Visualize Substation:
Diagram of the network, which can be overlaid on a street map or viewed as a chart or list. On the diagram, you can left or right click to see network connectivity or asset details.
Report Faults:
Report faults from breakers, AFS switches, tripsavers, and FCIs
Fault Current Indicators:
Dashboard showing data quality and field work needed on installed FCI
High Voltage Transformers:
Report transformers that are shorting out, based on those showing high voltage at downstream meters
Monitor Data Movement:
Manage the data flow from transmission and distribution to reporting systems in order to catch reporting errors

Technical Notes

All code is written in 2015 specifically for this project. The back end that touches legacy systems is written in Grails/Groovy/Tomcat, everything else is JavaScript (NodeJS/ Express 4/ MongoDB 3). All front-end code uses HTML5 and JS/Angular.

Web Server/Frameworks

The system on OpenShift is simplified for demo purposes and uses extracts to MongoDB from other databases (Postgres, SQL Server, Oracle, DB2) so only needs:

  • Apache server for the front end
  • NodeJS/Express/MongoDB for back end (REST API)

This was largely developed in Windows 8 locally. NodeJS, Tomcat, and MongoDB are on Red Hat Linux in the production environment.

Databases

The data is stored in sevaral databases; Postgres (meter and light events), Oracle (assets and work), SQL Server (aggregated summaries), eDNA (real time data), and MongoDB (topology).

Front-End Components

  • AngularJS: validation, resource, animation, ui.router, ui.bootstrap
  • Angular third-party contributions: spinkit, csv, sanitize.
  • Angular custom directives for D3 and right-click.
  • D3 for visualization
  • Leaflet for mapping
  • Bootstrap CSS
